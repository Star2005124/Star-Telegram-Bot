# Waguri Kaoruko Telegram Bot

A feature-rich Telegram bot with:
- Inline menu (20 features)
- Owner-only settings (bot name, menu image, prefix, etc.)
- Configurable via commands

## Setup
1. Clone the repo
2. Run `npm install`
3. Set environment variables `BOT_TOKEN` and `OWNER_ID` (or edit `bot.js` for local testing)
4. `npm start`

## Environment Variables
- `BOT_TOKEN` – Your Telegram bot token from @BotFather
- `OWNER_ID` – Numeric Telegram user ID for the owner (default: 8058995701)
